## 管理资源

>webpack 根据正则表达式，来确定应该查找哪些文件，并将其提供给指定的 loader。示例中，所有以 .css 结尾的文件，都将被提供给 style-loader 和 css-loader

依赖包：

 - style-loader 和 css-loader
 - file-loader
 - csv-loader 和 xml-loader
